# Etherlink Shadownet Deployment Artifacts

This directory stores Etherlink Shadownet Testnet deployment artifacts only.

No mainnet deployment artifacts belong here. Do not commit private keys, seed phrases, funded deployer keys, API tokens, or secret RPC URLs.

## Files

- `preflight-report.md`: latest local preflight report.
- `latest.json`: latest draft or operator-recorded deployment artifact.
- `latest.md`: human-readable latest summary.
- `runs/`: timestamped Shadownet run artifacts.
- `test-markets/`: test market config snapshots.

All artifacts must state that they are Shadownet testnet only and not production-ready.
