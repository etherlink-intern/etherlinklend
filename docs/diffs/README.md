# Fork Diff Records

Fork diffs belong here.

Each diff record MUST include:

- Upstream repository.
- Upstream commit SHA.
- Local commit SHA.
- Diff command.
- Files changed.
- Classification of each change.
- Tests mapped to each change.
- Protocol engineer reviewer.
- Security engineer reviewer.
- External audit reference if production-impacting.

Protocol logic diffs MUST NOT be released to mainnet without external audit.
